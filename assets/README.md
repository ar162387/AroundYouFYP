# Assets Directory

This directory contains the app's static assets like images and icons.

## Required Assets

The following placeholder files should be replaced with actual images:

- `icon.png` - App icon (1024x1024)
- `splash-icon.png` - Splash screen icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

For now, the app will use emojis and gradients for visual elements.
